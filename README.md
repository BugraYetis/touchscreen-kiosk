# 🖥️ Touchscreen Kiosk UI

> Fuar ve etkinlikler için özel dokunmatik ekran arayüzü

## 🌐 Demo
**[Canlı Demo](https://mae.com.tr/touchscreen%202/)**

## ✨ Özellikler
- Touch API ile dokunmatik ekran optimizasyonu
- GSAP ile akıcı geçiş animasyonları
- Kiosk modunda tam ekran deneyimi
- Ürün/hizmet tanıtım kartları
- Gesture & swipe desteği

## 🛠️ Teknolojiler
`Vanilla JavaScript` `Touch API` `GSAP` `CSS3`
